
### Wordpress Seed Theme

This is a simple seed theme with Bower and gulp interactions.

	npm install
	bower install

And you should be good to go.

#### Javascript libraries included

* jQuery
* Bootstrap Sass
* Lightgallery
* Fontawesome
* Imagesloaded
* Slick Carousel
* Respond.js
* Enquire.js
* MatchHeight
* Cycle 2
* Isotope
